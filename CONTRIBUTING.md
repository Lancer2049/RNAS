# Contributing to RNAS-OpenWrt

Thank you for your interest in contributing!

## Ways to Contribute

1. **Report Issues** - Bug reports, feature requests
2. **Test Protocols** - Help test untested protocols
3. **Improve Documentation** - Fix typos, add examples
4. **Code Contributions** - Scripts, configurations, tools

## Testing Status

We need help testing these protocols:
- [ ] IPoE (DHCP+) - Config ready, needs testing
- [ ] L2TP - Config ready, needs testing
- [ ] PPTP - Config ready, needs testing
- [ ] SSTP - Config ready, needs testing

## Pull Request Guidelines

1. Test your changes
2. Update documentation if needed
3. Follow existing code style
4. Include test evidence

## Testing Evidence

When testing, please provide:
- Test commands used
- Output/logs
- Screenshots if applicable
- Environment details
